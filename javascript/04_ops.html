JavasScript operators

Literal values and variables can be combined with mathematical operaters (+, -, *, /) and be assigned to a variable with the assignment operator (=). 



The modulus operator (%) returns the reminder of a divison





The operater (++) increments a variable and the operator (--) decrements a variable.







Assignment operators

The combination  var (op)(=) value is called the assignment operator. It uses the variable itself as the first operand. For example: myLongVariable += 3 is a shortcut for myLongVariable = myLongVariable + 3
