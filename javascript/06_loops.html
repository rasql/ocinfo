Loops

A loop allows to repeat a certain sequence of instructions. The first statement (i=0) is the initialization. It is executed only once at the beginning. The second statement (i<7) is the termination condition. It is at the beginning of every iteration. Finally the last part (i++) is the incrementation. It is done at the end of each iteration. 



Try to modify the beginning and end values and the iteration.





You can also use the loop counter as an index for a string variable.





The for/in statement can be used to loop through the properties of an object.
