Bitwise operations

Internally integer numbers are represented as binary numbers, as sequence of zero and ones. The fonction toString(2) allows to transform a number into a string of base 2 




Right-shift

The >> operator shifts the bits a certain amount to the right. A right-shift by 1 correpsonds to a divison by 2. A right-shift by 2 correpsonds to a divison by 4. 







Left-shift

The << operator is a left-shift and corresponds to the multiplication by 2.






Bitwise AND

The & operator combines each individual bit of the two operands






Bitwise OR

The | operator combines each individual bit of the two operands






Bitwise XOR (^)

The XOR operator returns 1 if the bits are different and returns 0 if the bits are identical. The space caracter in the last print instruction is added in order to right-align the three numbers.
