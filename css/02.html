CSS Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.

Border style

The border-style property specifies what kind of border to display. The following values can be used: dotted, dashed, solid, double,

Border width

The border-width property specifies the width of the 4 borders: top, right, bottom and right.
